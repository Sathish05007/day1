CSS Introduction, Types & Selectors

1. What is CSS?

CSS (Cascading Style Sheets) is used to style HTML elements, control layout, colors, fonts, and more. It separates content (HTML) from presentation (design).

2. Types of CSS

A. Inline CSS

This is inline CSS

B. Internal CSS

This text is styled using internal CSS (blue color for all <p> elements).

C. External CSS

External CSS is written in a separate .css file and linked with <link>. (Not shown here, but works like internal CSS.)

3. CSS Selectors

1. Universal Selector (*)

Removes default margin/padding for all elements.

2. Element Selector

Example: p { color: blue; } changes all paragraph text to blue.

3. ID Selector (#)

This heading is styled with an ID selector

4. Class Selector (.)

This paragraph uses the "highlight" class.

5. Group Selector (,)

Example: h1, h2, p { font-family: Arial; } applies the same style to multiple tags.

6. Descendant Selector (space)

This paragraph is inside a div (descendant selector applied - green text).

7. Child Selector (>)

This span is a direct child of a div (purple text).
